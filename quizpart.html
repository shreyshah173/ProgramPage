SQL SELECT Statement
Suppose a database has a table named Employees with these fields: id, name, email, and city.
How can we select the email addresses of all the employees from the Employees table?
  • Option A:
    SELECT id, email FROM Employees;
  • Option B:
    SELECT city FROM Employees;
  • Option C:
    SELECT email FROM Customers;
  • Option D:
    SELECT email FROM Employees;
